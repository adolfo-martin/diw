Tipos de posicionamiento

Posicionamiento estático. position: static

Es el posicionamiento por defecto

Cada etiqueta aparece a continuación de la anterior

Hay que tener en cuenta si la etiqueta es de tipo bloque o de tipo en línea

Posicionamiento fijo

position: fixed

El contenido aparece en las coordenadas indicadas siempre

Posicionamiento absoluto

position: absolute

El contenido aparece en las coordenadas indicadas

Posicionamiento relativo

position: relative

La etiqueta físicamente ocupa el espacio que ocupaba con posicionamiento estático.

Y la puedes mover de esa localización.
